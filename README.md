# Brilho Certo

Landing page e MVP de agendamento para faxina residencial.

## Recursos

- Landing page responsiva com identidade visual da Brilho Certo.
- Calendario mensal com dias livres, ocupados, indisponiveis e passados.
- Formulario com validacao, mascara de WhatsApp e mensagem personalizada.
- Painel administrativo simples para bloquear/liberar dias e gerenciar reservas.
- Favicon, apple icon e imagem de compartilhamento configurados.

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Validacao

```bash
npm run lint
npm run build
```

## Deploy

Projeto conectado ao Vercel com deploy automatico via GitHub.
